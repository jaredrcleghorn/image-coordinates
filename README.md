# image-coordinates

## Build Image

```shell
./scripts/build.sh
```

## Start Container

```shell
./scripts/run.sh
```

## Run Tests

```shell
./scripts/test.sh container-name
```

## Examples

```shell
./scripts/example-small.sh
./scripts/example-large.sh
```

## Stop Container

```shell
./scripts/stop.sh continaer-name
```

## Remove Image

```shell
./scripts/remove.sh
```
